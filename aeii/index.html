This is the official website for Project AEII. The project itself is a remastered version of the game Ancient Empires II from Symbian time and provides supports for Desktop, Android and iOS platfroms.

New features

  • New tiles
  • New & renewed units
  • New abilities
  • New & renewed status
  • New campaigns
  • In-game map editor
  • Online map sharing
  • Online multiplayer mode
