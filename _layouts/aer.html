This is the official website for Project AEII (also known as Ancient Empires Reloaded). The project itself is a remastered version of the game Ancient Empires II from Symbian time and provides support for Desktop, Android and iOS platfroms. You can find more information about the game in our forum.

Haven't played the original game before and want some look into the game? Ancient Empires Reloaded is reviewed in Appliv, take a look!

Desktop shortcut keys

  • A - Attack
  • G - Move
  • B - Recruit
  • H - Heal
  • S - Summon
  • C - Repair/Occupy
  • Space - Standby
  • Right-click - Cancel actions
  • Esc - Close dialogs (Only available when you are in game)
  • Enter - Open message box/send message (Only available in multiplayer mode)
